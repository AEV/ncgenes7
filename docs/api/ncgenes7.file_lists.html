class ncgenes7.file_lists.generic.MatchSubStringFileList(*, split_char='_', match_indices=-1, directory_matcher_depth=-1, **file_list_kwargs)[source]¶

Bases: nucleus7.data.file_list.FileListExtendedMatch

File list that first subtracts suffix and prefix and then can split the rest using a split_char and select splits on indices and joins them with same split_char to build the matching prefix

Parameters

  • split_char – character in the file name to split with

  • match_indices – indices of the split name to use for matching

  • directory_matcher_depth – if is >= 0, then directory of this depth starting from the file name will be used, e.g. d2/d1/d0/fname will result to d1 will be added to match pattern if directory_matcher_depth = 0

match_fn(path, key)[source]¶

This method will be used inside of self.match()

It uses basename without extension and removes suffixes from self.matching_suffixes and prefixes from self.matching_prefixes

Parameters

  • path – complete path of the file

  • key – key from file_names using for to create this pattern

Returns
