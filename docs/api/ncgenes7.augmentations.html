class ncgenes7.augmentations.image.ImageRandomCutout(min_cut_length=0.01, max_cut_length=0.05, is_normalized=False, **augmentation_kwargs)[source]¶

Bases: ncgenes7.augmentations.image._ImageRandomCutout

Set a random rectangle in the image to zero

Parameters

  • min_cut_length – Minimum rectangle length in each direction relative to image size, e.g. in [0, 1]; if single int, then it will be the same for all dimensions

  • max_cut_length – Maximum rectangle_length in each direction relative to image size, e.g. in [0, 1]; if single int, then it will be the same for all dimensions

  • is_normalized – Whether or not the image is already normalized, e.g. its mean = 0

incoming_keys¶

  • images : image or list of images, [h, w, num_channels], tf.float32

generated_keys¶

  • images : image or list of images, [h, w, num_channels], tf.float32

References

DeVries, T., & Taylor, G. W. (2017). Improved regularization of convolutional neural networks with cutout. arXiv preprint arXiv:1708.04552.
