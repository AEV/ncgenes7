class ncgenes7.plugins.utils.reshape.ConcatPlugin(axis=-1, **plugin_kwargs)[source]¶

Bases: nucleus7.model.plugin.ModelPlugin

Plugin to concat list inputs.

It will take all inputs to it, which must be lists, e.g. {input1: [data1, data2, data3]} and concat it over the axis for this key

Parameters

axis – axis to concat the inputs; can be a dict with mapping of the input key (also nested with ‘//’ separator) or a int, which will be then the same axis for all inputs; in case of axis mapping, ‘default’ key may exist and will be used if the key was not found inside of the axis mapping

class ncgenes7.plugins.utils.reshape.ExtractDimensionsFromBatch(num_dims_in_batch=None, **plugin_kwargs)[source]¶

Bases: nucleus7.model.plugin.ModelPlugin

Plugin extracting additional dimensions from the batch size

There are possible ways to provide needed shape:

  • explicitly provide shape_in_batch (tensor with dimensions to extract from batch) to shape_in_batch key

  • provide a tensor, which has already the first dimensions as needed to tensor_with_shape_in_batch and provide num_dims_in_batch arg

Parameters

num_dims_in_batch – number of dimensions inside of batch, if tensor to infer the shape was provided

incoming_keys¶

  • features : original tensor

  • shape_in_batch : tensor specifying which dimensions have been hidden

  • tensor_with_shape_in_batch : tensor with the shape that must be unmasked up to num_dims_in_batch

generated_keys¶

  • features : the input tensor with additional dimensions restored

generated_keys = ['features']
incoming_keys = ['features', '_shape_in_batch', '_tensor_with_shape_in_batch']
